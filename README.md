# Full-Stack E-Commerce Demo

A full-stack e-commerce application built with **Next.js**, **FastAPI**, and **PostgreSQL (Neon)**.

**Live Demo:**  
Frontend → https://fullstack-ecommerce-site-tawny.vercel.app/  
Backend API → https://ecommerce-api-kmdc.onrender.com/

## Features

- **Product browsing** (grid, detail pages)
- **Shopping cart**
  - Add / remove items
  - Update quantity
  - Persistent via `localStorage`
- **Checkout**
  - Email confirmation step
  - Portfolio demo only (no real payments)
- **Backend API**
  - FastAPI
  - PostgreSQL + SQLAlchemy
- **Deployment**
  - Frontend → Vercel  
  - Backend → Render  
  - Database → Neon

## Tech Stack

### Frontend
- Next.js 14
- React
- TypeScript
- Tailwind CSS
- Fetch API

### Backend
- FastAPI
- SQLAlchemy
- Pydantic v2
- Uvicorn
- psycopg2-binary

### Database
- PostgreSQL (Neon)
